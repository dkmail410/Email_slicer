# Email Slicer 📧

A simple and efficient Python script that takes an email address as input and slices it to extract the **username** and the **domain name**.

---

## 🚀 How It Works
The script uses Python's string manipulation methods to identify the `@` symbol and split the address into two distinct parts.

1. **Input:** The user enters an email (e.g., `developer@example.com`).
2. **Processing:** The script strips any extra whitespace and partitions the string.
3. **Output:** It displays the identified username and domain.

## 🛠️ Features
* **User-Friendly:** Simple command-line interface.
* **Clean Data:** Automatically removes leading/trailing spaces from the input.
* **Educational:** A great example of string slicing and formatting in Python.

## 💻 Usage
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/dkmail410/Email_slicer.git](https://github.com/dkmail410/Email_slicer.git)
    cd Email_slicer
    ```
2.  **Run the script:**
    ```bash
    python email_slicer.py
    ```
3.  **Example:**
    * **Input:** `jane.doe@gmail.com`
    * **Result:** > Your username is 'jane.doe' and your domain is 'gmail.com'

## 📜 Requirements
* Python 3.x
